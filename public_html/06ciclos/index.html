Ciclos

Este tutorial introduce al estudiante al uso de ciclos para realizar tareas repetitivas de manera automática sobre un conjunto de datos.

Sentencia repetitiva de código

Los ciclos, bucles o sentencias repetitivas son fragmentos de código que se ejecutan varias veces hasta que la condición asignada a dicho ciclo sea cumplida, los dos tipos de bucles más utilizados en programación son el ciclo while y el ciclo for.

El primer ciclo, while se define usando la palabra reservada while de la siguiente manera:

while([Condición]) { [Lista de instrucciones separadas por ejecutar] }

De esta manera el ciclo while funciona como un condicional que permite ejecutar un bloque de código repetidas veces hasta que el condicional deje de satisfacerse.

Como ejemplo, suponga que definimos e inicializamos una variable contar con valor 10, y otra variable texto en "".

var contar = 10;
var texto = "";

Ahora vamos a escribir un programa que imprima de 1 a contar el texto "El número es: numero_actual". Para esto debemos ajustar una variable iterador que itere desde 1 a contar para que se cumpla la condición de iterador < contar, de esta manera se recorren todos los números desde 1 hasta 10. 

contar = 10;
var iterador = 1;
while(iterador <= contar){
    texto += "<br>The number is " + iterador;
    i ++;
}

El Segundo ciclo, for se define usando la palabra reservada for de la siguiente manera:

for([variable_de_recorrido ; condicion_de_variable; incremento_de_variable_de_recorrido ]) { [
    Lista de instrucciones separadas por ejecutar] 
}

De esta manera el ciclo for funciona con una estructura definida por un condicional, en este caso el condicional se define por medio de una variable que va a hacer un recorrido en un rango definido, y mediante un condicional se va a definir el punto de finalizacion y partida asi como el incremento (positivo o negativo) de esta variable.

Como ejemplo, suponga que definimos e inicializamos un arreglo marcas, el cual va a contener una lista de marcas reconocidas de ropa y accesorios, por lo cual queremos listarlos para visualizar todos en una lista.

var marcas = ["Gucci", "D&G", "Adidas", "Puma", "Ray-Ban", "Polo Club"];

Ahora vamos a escribir un programa que imprima todos los elementos de la lista marcas para exibirlos en una lista que se mostrara al usuario.
