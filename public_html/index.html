ipp5

ipp5 es un proyecto que busca motivar e introducir la programación a jóvenes estudiantes de manera divertida y didáctica.

ipp5 usa un lenguaje de programación llamado Java Script y una librería multimedia (gráficos y sonidos) denominada p5.js para introducir los conceptos básicos de la programación de computadores.
